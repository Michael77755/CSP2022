My project is called the “Minion Creator”. The main objective as in the name is to create a minion based off of the user’s inputs. The code allows the user to have 18 different variations of a minion such as a small minion with one eye and red overalls all the way to a large minion with two eyes and blue overalls. Once the user finishes customizing their minion, the minion gets on top of Gru’s car and rides away off the screen. Then, the user is greeted by their minion with a background saying TA-DA!


My project is called “Tic Tac Toe”. The point of the game is to just get three minions in a row, whether it is diagonal, vertical or horizontal. When either player_x/the yellow minion or player_o/the purple minion wins it is displayed at the bottom with either the program saying, "Player X wins" or "Player O wins".


My project is called “Stuffed Fun”. This project is an electronic card for a childrens hospital. When you start the program the card plays music and has a spiderman moving left to right, when you click the spiderman four stuffed animals appear with four differenet encouraging messages.


My project is called "Snake Game". The point of the game is to collect as many bananas as you can. You are able to change costumes by left clicking anywhere in the game. If you are to run into the wall the the game ends and you lose.
